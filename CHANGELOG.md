# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial open-source release
- WPF desktop application with MVVM architecture
- Angular web application
- RESTful API with ASP.NET Core
- JWT-based authentication
- Role-based access control (SuperAdmin, Moderator, Cashier)
- Product and category management
- Stock movement tracking
- Reporting functionality (stock reports, low-stock alerts, movement history)
- Input validation with FluentValidation
- Structured logging with Serilog
- Unit tests for core services
- Clean Architecture implementation
- Multi-language support (English, Arabic)
- Local date/time conversion for web app
- Comprehensive documentation

### Security
- JWT authentication
- Password hashing (SHA256)
- Input validation on all API endpoints
- CORS configuration
- Security best practices documentation

## [1.0.0] - 2024-11-13

### Added
- Initial release
- Core inventory management features
- Dual UI support (WPF + Angular)
- API with Swagger documentation
- Role-based permissions system
- Comprehensive test coverage
